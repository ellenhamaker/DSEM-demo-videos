Introduction

Dynamics structural equation modeling (DSEM) is based on a combination of time series modeling, multilevel modeling, and structural equation modeling. It can be used to analyze time series data and intensive longitudinal data, such as obtained with daily diaries, ecological momentary assessments, experience sampling, ambulatory assessments, and other techniques that tend to result in (relatively) large numbers of repeated measures (say, more than 20) per case.

More information about DSEM and how it was implemented in the software package Mplus can be found here: https://ellenhamaker.github.io/DSEM-book-chapter/ and here: https://www.statmodel.com/TimeSeries.shtml.

\(~\)

DSEM models

The DSEM demo videos show how to use Mplus to specify specific models, and how to interpret the output. In addition, there are also videos that focus on various aspects related to DSEM analyses, such as Bayesian estimation and plotting options.

\(\triangleright\) Multilevel model

In the DSEM demo videos on multilevel DSEM models, these data are used:

  • multilevel data

The models that are run as part of the demo video part 1 are:

  • Model 1: random means, no random slopes, no random residual variances (Mplus input and output)

  • Model 2: random means, random slopes, no random residual variances (Mplus input and output)

  • Model 3: random means, random slopes, and residual variances that covary (Mplus input and output)

The models that are run as part of the demo video part 2 are:

  • Model 4: random means, slopes, residual variances which are indicators of a latent variable (Mplus input and output)

  • Model 5: random means, slopes, residual variances which are predicted by a level 2 predictor (Mplus input and output)

  • Model 6: random means, slopes, residual variances which are predicted by a level 2 predictor and that predict a level 2 outcome (Mplus input and output)
